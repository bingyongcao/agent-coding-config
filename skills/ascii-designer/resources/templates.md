# ASCII UI Templates

## Common Elements

### Window Frame
```text
+-------------------------------------------------------+
|  Window Title                                     [x] |
+-------------------------------------------------------+
|                                                       |
|                                                       |
+-------------------------------------------------------+
```

### Components

**Buttons**
`[ OK ]`  `[ Cancel ]`  `[ < Back ]`  `[ Next > ]`

**Inputs**
Label: `[____________________]`
Search: `(o) ____________________`

**Tabs**
`[ Tab 1 ] [ Tab 2 ] [ Tab 3 ]`
`-----------------------------`
